h1. OscMachine

Simple sample player that plays samples triggered by OSC messages.

h2. Installation

h3. Without git

Download archive from "here":http://github.com/lokijuhz/OscMachine/downloads, extract and copy OscMachine.sc and OscMachine.html to Extension folder. (On Mac: @~/Library Application Support/SuperCollider/Extensions@).

h3. With git

Clone repository:
@git clone git://github.com/lokijuhz/OscMachine.git@.

h2. Usage

Needs RedSampler Quark.

Further information see help file.
